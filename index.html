How it’s calculated (simple version)
• Start with annual base rates for the zip code (defaults are BI 100/300, PD 100k, SLL 300k, FPB 10k, Comp/Coll 500 deductibles).
• Apply a liability limit factor, PD scaling, and a tort multiplier.
• Scale Comp/Coll using vehicle value (heuristic).
• Convert to 6‑month or 12‑month display.
